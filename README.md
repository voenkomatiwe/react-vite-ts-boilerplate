# React Vite TypeScript Boilerplate

This is a boilerplate project using **React, Vite, and TypeScript**, preconfigured with **Tailwind CSS, shadcn/ui, ESLint, Prettier, and Husky** for a smooth development experience.

## Features

- ⚡ **Vite** – Fast development server and production builds.
- ⚛️ **React 19** – Latest version with modern features.
- 🛠 **TypeScript** – Static type checking.
- 🎨 **Tailwind CSS** – Utility-first CSS framework.
- 🏗 **shadcn/ui** – Beautiful, accessible, and customizable UI components.
- ✅ **ESLint & Prettier** – Linting and code formatting.
- 🐶 **Husky & lint-staged** – Pre-commit hooks for better code quality.
